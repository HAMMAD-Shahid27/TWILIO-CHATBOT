Quick Start Guide

  1. Clone the repository: git clone https://github.com/yourusername/llm-twilio-callbot.git
  2. Install dependencies: pip install -r requirements.txt
  3. Copy environment file: cp env.example .env
  4. Configure your API keys in .env
  5. Run the application: python app.py

  1. Create a Twilio account and get a phone number
  2. Set up ngrok: ngrok http 5000
  3. Configure webhook URL in Twilio console
  4. Set webhook to: https://your-ngrok-url.ngrok.io/webhook

Customize the bot personality in config/settings.py:

BOT_PERSONALITY = {
    "name": "Your Bot Name",
    "tone": "friendly and professional",
    "specialties": ["customer service", "support"]
}
